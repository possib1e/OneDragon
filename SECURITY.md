# Security Policy

OneDragon is intended for authorized security testing, white-hat research, and education only.

## Responsible Use

- Use this project only against assets you own or have explicit written permission to test.
- Keep scan scope narrow and documented before running automated workflows.
- Avoid running high-volume scans against third-party infrastructure without authorization.
- Review tool configuration before use, especially masscan, nmap, ffuf, xray, and AWVS-related settings.

## Reporting Security Issues

If you find a vulnerability in this project itself, please open a private report or contact the maintainer before publishing details. Include:

- A short description of the issue.
- Steps to reproduce.
- Expected and actual behavior.
- Impact and suggested mitigation, if known.

Please do not include real third-party target data in public issues.
