# v0.1.1 Maintenance Release Notes

This maintenance release focuses on project hygiene and contributor-facing documentation.

## Highlights

- Added installation notes for the legacy scanner workflow.
- Added configuration notes and `config.example.yaml` to document the intended config surface.
- Added a maintenance roadmap with near-term and medium-term goals.
- Improved CLI help by switching the main entrypoint to `argparse`.
- Deferred scanner module imports until after argument parsing so `--help` works before optional scanner dependencies are installed.
- Kept the responsible-use boundary visible in the README and security policy.

## Verification

```bash
python3 -m compileall start.py module
python3 start.py --help
```

## Next Steps

- Move hard-coded scanner paths into a validated config loader.
- Add a lightweight CI workflow for syntax checks.
- Split scanner wrappers into smaller testable units.
